# Rollup React Boilerplate

## Usage
- ``git clone https://github.com/GerardoGallegos/react-rollup-boilerplate.git``
- ``npm install``
- ``npm run build:dev``

## Dev Dependencies
* rollup
* babel
* size-limit
* standard

## Peer Dependencies
* react
* react-dom
* prop-types
* styled-components

## Comands

#### npm run build:dev
#### npm run build:prod
#### npm run size
