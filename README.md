# Rollup React Boilerplate

## Usage
- ``git clone https://github.com/GerardoGallegos/react-rollup-boilerplate.git``
- ``npm install``
- ``npm start``

## Dev Dependencies
* rollup
* babel
* size-limit
* standard
* jest

## Peer Dependencies
* react
* react-dom
* prop-types

## Comands

#### npm start
#### npm run build
#### npm run size
#### npm run test
#### npm run test:w
